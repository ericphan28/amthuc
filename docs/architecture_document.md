# Tài liệu Kiến trúc

## Kiến trúc tổng quan
- Sử dụng mô hình Client-Server
- Frontend giao tiếp với Backend thông qua RESTful API

## Các thành phần chính
- **Client**: React.js, Redux
- **Server**: Node.js, Express
- **Database**: MongoDB
